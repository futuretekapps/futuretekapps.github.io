It helps us track anonymized usage data such as which features are most popular, game completion statistics (like difficulty and time taken), and overall user engagement. This data is aggregated and does not personally identify you. It is used solely to improve the Application.

When the app crashes, this service collects information about the device state and the code that caused the crash.
